Limpieza de Datos

Manejar valores faltantes, tipos incorrectos, texto sucio y duplicados.

Valores faltantes

df.dropna() # Remove rows with NaN df.dropna(subset=['col']) # Only if 'col' is NaN df.fillna(0) # Replace NaN with 0 df.fillna(df['col'].median()) # Replace with median df['col'].interpolate() # Interpolate

Conversion de tipos

df['col'] = df['col'].astype(int) df['col'] = df['col'].astype(float) df['date'] = pd.to_datetime(df['date'])

Limpieza de texto

df['col'] = df['col'].str.strip() # Remove whitespace df['col'] = df['col'].str.lower() # Lowercase df['col'] = df['col'].str.replace(',', '') # Remove char df['col'] = df['col'].str.title() # Title Case

Duplicados y columnas

df.drop_duplicates() # Remove exact dupes df.drop_duplicates(subset=['col']) # Based on column # Rename df.rename(columns={'old': 'new'}) df.columns = ['col1', 'col2', 'col3'] # Rename all # Drop columns df.drop(columns=['col1', 'col2'])

Seleccion y Filtrado

Seleccionar columnas, filas y aplicar condiciones booleanas.

Seleccionar columnas y filas

df['col'] # Single column (Series) df[['col1', 'col2']] # Multiple columns (DataFrame) df.loc[0] # By label/index df.iloc[0:5] # By position (first 5) df.loc[df['col'] > 10] # By condition

Filtros booleanos

df[df['age'] > 18] # Single df[(df['age'] > 18) & (df['city'] == 'NYC')] # AND df[(df['age'] > 18) | (df['city'] == 'NYC')] # OR df[~(df['age'] > 18)] # NOT df[df['city'].isin(['NYC', 'LA'])] # In list

Query y ordenamiento

# Query syntax (cleaner) df.query('age > 18 and city == "NYC"') # Sorting df.sort_values('col') # Ascending df.sort_values('col', ascending=False) # Descending df.sort_values(['col1', 'col2']) # Multiple

GroupBy y Agregacion

Agrupar datos por categorias y calcular estadisticas por grupo.

GroupBy basico

df.groupby('category')['value'].mean() df.groupby('category')['value'].sum() df.groupby('category')['value'].count()

Agregaciones multiples

# Multiple agg functions on one column df.groupby('category')['value'].agg([ 'mean', 'median', 'std', 'count' ]) # Different functions per column df.groupby('category').agg({ 'value1': 'mean', 'value2': 'sum', 'value3': ['min', 'max'] })

Tablas de frecuencia y pivot

# Value counts df['col'].value_counts() # Frequency df['col'].value_counts(normalize=True) # Proportions # Pivot table pd.pivot_table(df, values='value', index='row_var', columns='col_var', aggfunc='mean' ) # Cross tabulation pd.crosstab(df['col1'], df['col2'])

Medidas de tendencia central, dispersion y correlacion.

Tendencia central

df['col'].mean() # Mean (average) df['col'].median() # Median (middle value) df['col'].mode() # Mode (most frequent)

Dispersion

df['col'].std() # Standard deviation df['col'].var() # Variance df['col'].min() # Minimum df['col'].max() # Maximum df['col'].quantile(0.25) # Q1 df['col'].quantile(0.75) # Q3

Correlacion y resumen

# Correlation df['col1'].corr(df['col2']) # Between two columns df.corr() # Correlation matrix # Full summary df.describe() # Numeric stats df.describe(include='all') # Include non-numeric

Histograma y dispersion

import matplotlib.pyplot as plt # Histogram plt.figure(figsize=(10, 6)) plt.hist(df['col'], bins=30, color='steelblue', edgecolor='white') plt.xlabel('Value') plt.ylabel('Frequency') plt.title('Distribution of Values') plt.show() # Scatter plot plt.scatter(df['x'], df['y'], alpha=0.5) plt.show()

Subplots y guardar
