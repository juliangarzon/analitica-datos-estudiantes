CONTEXT

Two tools, two roles

Plotly

Creates interactive charts with hover, zoom, legend toggles and clean web-ready visuals.

Line Bar Scatter Box

Streamlit

Wraps the analysis into a usable application with filters, layout, metrics and tables.

Sidebar Widgets Metrics Tabs

Mental model

Plotly makes the charts interactive. Streamlit makes the whole analysis interactive.

PATTERN

The reusable dashboard pattern

load data rename and clean columns create sidebar filters filter the dataframe compute KPIs build charts optionally show the filtered table

This is the part you should learn deeply. Once you understand this pattern, you can reuse it with many public datasets.

CHECKLIST

Minimum app requirements for today

3 filters

They must support comparison, not decoration.

3 KPIs

They must summarize the current selection.

3 charts

Each one should answer a different analytical question.
