Detecting Missing Values

The Code

# Check for missing values (count the missing socks) # How many missing per column? df.isnull().sum() # Percentage missing per column (df.isnull().sum() / len(df)) * 100 # Total missing in entire dataset df.isnull().sum().sum() # Rows with ANY missing value df[df.isnull().any(axis=1)]

What It Does

isnull() returns True/False for each cell

.sum() counts True values (missing)

.any(axis=1) checks if ANY column in a row is missing

Pro tip: Always check missing values FIRST before any analysis!

Handling Missing Values

Option 1: Remove (dropna)

# Drop rows with ANY missing df_clean = df.dropna() # Drop rows missing specific column df_clean = df.dropna(subset=['grade']) # Drop columns with missing df_clean = df.dropna(axis=1)

Use when: Missing is rare (<5%) or row is useless without the value

Option 2: Fill (fillna)

# Fill with a specific value df['grade'].fillna(0) # Fill with column mean df['grade'].fillna(df['grade'].mean()) # Fill with previous value df['grade'].fillna(method='ffill')

Use when: You need to keep all rows and have a reasonable replacement

Decision Guide: If >50% missing in a column, consider dropping the column. If <5% missing, dropping rows is usually safe.

Converting Data Types

The Problem

Numbers stored as text cannot be used for math:

# This fails if 'price' is text! df['price'].mean() # Error! # Check current types df.dtypes # price object (text!) # year object (text!)

The Solution

# Convert to integer df['year'] = df['year'].astype(int) # Convert to float (decimal) df['price'] = df['price'].astype(float) # Convert to datetime df['date'] = pd.to_datetime(df['date']) # Convert to category (saves memory) df['region'] = df['region'].astype('category')
Common Error: astype(int) fails if column has NaN values. Fill or drop NaN first!

GroupBy in Action

Basic GroupBy

# Count students by department df.groupby('department')['student_id'].count() # Average grade by department df.groupby('department')['grade'].mean() # Total enrollment by year df.groupby('year')['students'].sum()

Advanced GroupBy

# Group by MULTIPLE columns df.groupby(['year', 'department'])['students'].sum() # Multiple aggregations at once df.groupby('department').agg({ 'students': 'sum', 'grade': 'mean', 'school_id': 'count' })
Pattern: df.groupby('CATEGORY')['VALUE'].AGGREGATION()

Filtering with Multiple Conditions

Single Condition

# Students with grade > 80 df[df['grade'] > 80] # Only Bogota df[df['city'] == 'Bogota'] # Exclude nulls df[df['grade'].notna()]

Multiple Conditions

# AND: Both must be true df[(df['grade'] > 80) & (df['year'] == 2023)] # OR: Either can be true df[(df['city'] == 'Bogota') | (df['city'] == 'Medellin')] # IN: Match any in list cities = ['Bogota', 'Medellin', 'Cali'] df[df['city'].isin(cities)]
Common Mistake: Using and instead of &. Always use & (and) and | (or) in pandas!
